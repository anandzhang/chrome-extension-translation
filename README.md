# chrome-extension-translation
A chrome browser translation extension.

## Usage

https://urlc.cn/pDh2H3 点击下载压缩包，或者直接克隆项目。

解压之后，进入扩展程序管理界面 [chrome://extensions/](chrome://extensions/) ，然后打开开发者模式，接着加载解压后的文件夹。
